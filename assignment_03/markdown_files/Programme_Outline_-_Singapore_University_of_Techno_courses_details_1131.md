### Programme Outline



##### **Learning Objectives**

By the end of the course, participants should:

1. Know security testing techniques
2. Be able to identify the right testing techniques
3. Be able to apply state-of-the-art fuzzing tool

##### Day 1

* Intro to security
* Basics of testing
* Black-box Fuzzing
* Grey-box Fuzzing

##### Day 2

* Symbolic execution
* Concolic testing
* White-box Fuzzing
