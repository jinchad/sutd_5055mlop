### Programme Outline



##### **Learning Objectives**

* Introduction to Design Innovation
* Discover the user needs
* Define the main opportunities
* Develop ideas and Deliver tested conceptual ideas

##### Discover

* CN Analysis
* Contextual Needs
* Journey Mapping

##### Define

* Mission Statement
* Black Box
* Activity Diagram
* Affinity Analysis

##### Develop

* C-Sketch
* Word Tree
* Mindmapping

##### Deliver

* Prototyping
* User-Validation
* Scaled Prototypes
* Iterative Prototypes
